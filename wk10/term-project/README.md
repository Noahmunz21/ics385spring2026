# ICS 385 - Term Project 3 | Week 10
## Hawaii Hospitality Dashboard & Marketing Website

### Property: Maui B&B Collection
- **Island**: Maui
- **Property Type**: B&B (Bed & Breakfast)
- **Target Segment**: Local experience seekers (Eco-tourists, Honeymooners, Families, Adventure seekers, Beach lovers)

### Properties Seeded
1. Hana Hideaway B&B — Eco-tourists
2. Kaanapali Local Lodge — Beach lovers
3. Upcountry Maui Farmstay — Family travelers
4. Paia Surfer's Inn — Adventure seekers
5. Wailea Honeymoon Cottage — Honeymooners

### Technology Stack
- Node.js
- MongoDB Atlas
- Mongoose
- dotenv

### Setup Instructions
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file using `.env.example` as a template
4. Run `node seed.js` to seed the database
