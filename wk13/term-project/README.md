This week I built a fully interactive Visitor Statistics Dashboard for Hana Hideaway B&B using React, Chart.js via react-chartjs-2, and the OpenWeatherMap API. The dashboard features three visualizations — a bar chart of monthly Maui visitor arrivals sourced from DBEDT tourism data, a donut chart showing visitor origin breakdown, and three KPI metric cards — all driven by an island selector dropdown using React useState. Live weather data is fetched using useEffect and updates automatically when the island changes. In Weeks 14–15, I plan to add Passport.js admin authentication protecting the dashboard route, Helmet.js security headers, input validation with express-validator, and deploy the full application to Render. AI Tools Used: Claude (Anthropic) assisted with component structure, Chart.js configuration, and OpenWeatherMap integration.
